Clipboard Conqueror Bookmarklets

Make sure Clipboard Conqueror and your inference engine of choice are ready.

Drag links to your bookmarks bar to use them anywhere!

Click them here. Note that re sends the last copied text as the user query.

Bookmarklets are not a particularly secure thing, make sure you trust the source.

Refreshing the page clears any bookmarlet with executing code.

Clipboard Conqueror Settings.

These buttons will apply the described setting.

Help Assistant

|||h,re|

Copy your question about how to use Clipboard Conqueror and then click this one. I haven't tested it out much. Let me know.
System JSON

|||`json,no||

Sends the system prompts as a JSON object. No prevents sending to AI. Use it when you want to copy CC invocations without sending to AI.
System Partial

|||`partial,no||

Sends system prompts headed by "promptName:"
System Chat Turns

|||`markup,no||

Sends the system prompts as seperate chat turns, with the prompt name replacing "assistant" in the prompt format. Some models may ragequit, some love it.
System Default

|||`none,no||

Sends the system prompts as text blocks with no dividers or names. This is the default behavior.
Clipboard To History

|||`agi,no||

Will clear the history and start a new history with the last copied text as a message from user. Used for priming secondary query.
Silent History

|||`csrh,no||

Sends last copy into the end of the history but does not add user query or assistant response to history this turn. It might always send as user.
Clear History

|||`clearHistory||

Clears Clipboard Conqueror's chat history.
silentChat |||sc,re||
Sends the history wihout adding the new turn to the history. For multiple queries against the same history.

Active Mode Bookmarklets (5-minute timer)

These bookmarklets have a 5 minute timer that runs on the page you're viewing when you activate the bookmarklet. If you close or refresh the page the timer is ended without toggling off the mode.

The resend timer lives in the bookmarklet, not Clipboard Conqueror. |||on| Must be toggled off with another copy when used without the bookmark. The very end of the script, 300000, is the default time.
